Sentimen Emosi

Aplikasi analisis sentimen emosi menggunakan metode Multinomial Naive Bayes, serta ektraksi fitur N-Gram menggunakan Framework Django.



Tentang Web

Aplikasi ini dibangun untuk menyelesaikan salah satu syarat kelulusan mata kuliah Tugas Akhir. model yang digunakan pada aplikasi ini menggunakan metode Multinomial Naive Bayes dan fitur ekstraksi TF-IDF Vectorizer dengan N-Gram range (1, 2) atau unigram dan bigram.
