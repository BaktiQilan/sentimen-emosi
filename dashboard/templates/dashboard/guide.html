Alur Proses pada Menu Modeling

Pertama, Get New Data

Deskripsi:

1. Tombol untuk mendapatkan data dari Iteung

2. Tombol untuk mendapatkan data dari Twitter. tetapi harus disertai dengan memasukan keyword atau kata kunci.

3. Tombol untuk mendapatkan data dari file yang diupload. *Catatan: file harus berupa txt hasil dari export WhatsApp.

4. Tombol untuk menampilkan modal yang berisis step-by-step cara export history chat WhatsApp.

5. Table yang akan menampilkan data.

Deskripsi:

4. Tombol untuk memasukan data kedalam dataset pada database dan akan langsung melakukan training dan testing ulang pada model.
